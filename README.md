# R-Ops Demo - Event and Alert Management Interface

A beautiful event and alert management interface inspired by Jony Ive design principles, featuring a distinctive black and yellow color scheme.

## Overview

R-Ops Demo provides operations teams with streamlined workflows for monitoring, triaging, and responding to system events through an intuitive interface that transforms raw events into actionable alerts.

## Features

- **Alert Management**: Interactive alert dashboard with priority-based visual hierarchy
- **Event Management**: Comprehensive event log with filtering and search capabilities
- **Teams Management**: Team coordination with escalation schedules and on-call rotations
- **Rules Engine**: Visual rule builder for event-to-alert transformation logic
- **Analytics**: Performance metrics and reporting for operational insights

## Tech Stack

- React 19+ with TypeScript
- Vite for fast development and optimized builds
- Modern CSS with black/yellow design system
- Mock Service Worker for API simulation
